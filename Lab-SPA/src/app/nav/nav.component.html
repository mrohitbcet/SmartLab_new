Smart Pathalogy Lab

We provides pathology labs the flexible information module that is specifically used to disseminate and record the data corresponding to all the test performed in the laboratory. It facilitates users by providing the functionality of receiving online requests from doctors and also allows laboratory personnel to create requests.

Product Description
1)Book a test for the customer
2)Enter customer's details
3)Give test reports to the customer
4)Facility to download the report of test performed.
5)Facility to send test report to customer in email.
